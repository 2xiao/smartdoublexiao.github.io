219. Contains Duplicate II
思路：hashmap
答案：

public class Solution {

 public boolean containsNearbyDuplicate(int[] nums, int k) {

 Map<Integer, Integer> map = new HashMap<Integer, Integer>();

 for (int i = 0; i < nums.length; i++) {

 if (map.containsKey(nums[i])) {

 if (i - map.get(nums[i]) <= k) return true;

 }

 map.put(nums[i], i);

 }

 return false;

 }

}

 profit = Math.max(profit, max);

88. Merge Sorted Array
思路：已排序数组，应该从最大的数开始比较，而且--操作只能在较大的数被选取之后做。最后再检查nums2剩下的数。
答案：

public class Solution {

 public void merge(int[] nums1, int m, int[] nums2, int n) {

 int i = m - 1, j = n - 1, k = m + n - 1;

 while (i > -1 && j > -1) nums1[k--] = (nums1[i] > nums2[j]) ? nums1[i--] : nums2[j--];

 while (j > -1) nums1[k--] = nums2[j--];

 }

}

169. Majority Element
思路：投票算法，因为题目默认肯定会存在一个【n/2】次以上的数，所以只要把数组遍历一遍，不断更新count，最后使得count大于0的数一定是那个major。
答案：

public class Solution {

 public int majorityElement(int[] nums) {

 int count = 1, major = nums[0];

 for (int i = 1; i < nums.length; i++) {

 if (count == 0) {

 major = nums[i];

 count++;

 } else if (major == nums[i]) {

 count++;

 } else count--;

 }

 return major;

 }

}

189. Rotate Array
思路：先将全数组反转 ，再将1~k个数反转，再将k+1~n个数反转。
答案：

public class Solution {

 public void rotate(int[] nums, int k) {

 k %= nums.length;

 reverse(nums, 0, nums.length - 1);

 reverse(nums, 0, k - 1);

 reverse(nums, k, nums.length - 1);

 }

 public void reverse(int[] nums, int a, int b) {

 while (a < b) {

 int n = nums[a];

 nums[a] = nums[b];

 nums[b] = n;

 a++;

 b--;

 }

 }

}

414. Third Maximum Number
思路：
答案：

public class Solution {

 public int thirdMax(int[] nums) {

 long max = Long.MIN_VALUE, med = max, min = Long.MIN_VALUE;

 int count = 0;

 for (int x : nums) {

 if (x == max || x == med) {

 continue;

 }

 if (x > max) {

 min = med;

 med = max;

 max = x;

 count++;

 } else if (x > med) {

 min = med;

 med = x;

 count++;

 } else if (x >= min) {

 min = x;

 count++;

 }

 }

 return count >= 3 ? (int) min : (int) max;

 }

}

283. Move Zeroes
思路：for循环的另一种方式：for (int i : nums) 空间复杂度o(n)

public class Solution {

 public void moveZeroes(int[] nums) {

 if (nums == null || nums.length == 0) return;

 int n = 0;

 for (int num : nums) {

 if (num != 0) nums[n++] = num;

 }

 while (n < nums.length){

 nums[n++] = 0;

 }

 }

}

349. Intersection of Two Arrays
思路：Set<Integer> set = new HashSet<>();有待了解，但是总体思路就是把数组转Integer，利用Integer的方法进行比较然后又把结果赋值给数组。
答案：

public class Solution {

 public int[] intersection(int[] nums1, int[] nums2) {

 Set<Integer> set = new HashSet<>();

 Set<Integer> intersect = new HashSet<>();

 for (int i = 0; i < nums1.length; i++) {

 set.add(nums1[i]);

 }

 for (int i = 0; i < nums2.length; i++) {

 if (set.contains(nums2[i])) {

 intersect.add(nums2[i]);

 }

 }

 int[] result = new int [intersect.size()];

 int i = 0;

 for (Integer n : intersect) {

 result [i++] = n;

 }

 return result;

 }

}

453. Minimum Moves to Equal Array Elements
思路：一次移动n-1个数+1，相当于一次移动1个数-1,将所有数都减到最小数即可。
答案：

public class Solution {

 public int minMoves(int[] nums) {

 if (nums.length == 0) return 0;

 int min = nums[0];

 for (int n : nums) min = Math.min (n, min);

 int res = 0;

 for (int n : nums) res += n - min;

 return res;

 }

}

389. Find the Difference
思路：异或的用法，把所有的字符都异或起来，然后两两相消。
答案：

public class Solution {

 public char findTheDifference(String s, String t) {

 char c = 0;

 for (int i = 0; i < s.length(); ++i) {

 c ^= s.charAt(i);

 c ^= t.charAt(i);

 }

 c ^= t.charAt(t.length() - 1);

 return c;

 }

}
