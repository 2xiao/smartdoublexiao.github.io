组件的Refs：
Refs是访问到组件内部DOM节点唯一可靠的方法，Refs会自动销毁 对子组件的引用，不要在render或render之前对Refs进行调用，不要滥用Refs.

十二、React样式
css命名规范：dialog__confirm-button-highlight

注意样式的驼峰写法，注意class需要更改className：style = {styleComponentHeader.header} className="smallFontSize"
